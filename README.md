# odin-recipes

A small collection of static recipe pages created for The Odin Project Foundations exercises. Each recipe is a standalone HTML file located in the `recipes/` folder.

Project structure

- recipes/
  - casserole.html — Chilaquiles Breakfast Casserole
  - french_toast.html — Cinnamon French Toast Bake
  - cinnamon_rolls.html — Overnight Cinnamon Rolls
- index.html — (site index linking to recipes)
- README.md — this file
