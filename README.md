# Odin Recipes

A simple static website showcasing traditional Turkish recipes.

## Project Structure

```
index.html
recipes/
  adana.html
  manti.html
  sarma.html
```

## Recipes Included

- **Adana Kebap**: Spicy minced lamb skewers from Adana.
- **Kayseri Manti**: Turkish dumplings with yogurt and butter sauce.
- **Yaprak Sarma**: Grape leaves stuffed with rice and herbs.

## Usage

1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Click on any recipe to view details and instructions.
